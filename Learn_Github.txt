(A) HOW TO UPLOAD FILES/FOLDER::::
1) Create repository/ Github Account if new user
2) Create repository
3) or Inheritate other repository via link share with access
4) OPEN GIT BASP
5) TYPER THE COMMAND TO OPEN YOUR PARTICULAR DIRECTORY EXAMPLE:
   cd Desktop/SHAMBHU/Main_folder (in my case)
6) After open type
   git init 
   git add .
   git commit -m "I WANTS TO DOWNLOAD YOUR FILES"
   git branch -M origin main
   git push -u <paste link here>
 
done!


(B) HOW TO UPLOAD INDIVIDUAL FILES:::::

git add <filename with extension>   ENTER 
git commit -m "NEW FILE UPLOAD"
git push -u origin main
